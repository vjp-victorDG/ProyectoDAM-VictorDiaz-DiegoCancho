# 03 - Planificación del Proyecto

## 1. Fases del Proyecto
| Fase | Descripción | Duración Estimada |
| :--- | :--- | :--- |
| Fase 1 | Análisis y Diseño | [Ej: 1 semana] |
| Fase 2 | Configuración Infraestructura | [Ej: 2 semanas] |
| Fase 3 | Implementación y Pruebas | [Ej: 2 semanas] |
| Fase 4 | Documentación Final y Entrega | [Ej: 1 semana] |

## 2. Cronograma (Hitos principales)
* **Hito 1:** Finalización de documentos de análisis y diseño.
* **Hito 2:** Servidor web y base de datos operativos.
* **Hito 3:** Seguridad (Firewall/SSH) configurada y auditada.
* **Hito 4:** Entrega final del proyecto.

## 3. Distribución de tareas
* **Responsable de Gestión (Víctor/Diego):** Análisis, diseño, planificación, operaciones y plan de recuperación.
* **Responsable de Infraestructura (Víctor/Diego):** Backups, BBDD, monitorización, servidor web y firewall.
